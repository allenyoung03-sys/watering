隐私政策 — 植觉

最后更新日期：2026 年 5 月 9 日

引言

「植觉」（以下简称"本应用"）尊重并保护您的隐私。本隐私政策说明了我们如何收集、使用和披露您的信息。

如果您使用本应用，即表示您同意本隐私政策所述的内容。

1. 我们收集的信息

1.1 您主动提供的信息

信息类型说明
植物照片您拍摄或从相册选择的植物图片，用于植物识别和养护记录
个人资料您可选的昵称和头像（仅本地存储，不上传服务器）
养护记录您记录的浇水、施肥、修剪、除虫等养护操作日记

1.2 自动收集的信息

信息类型说明
本地用户标识应用首次启动时自动生成的 UUID，用于区分不同设备的数据（仅本地存储）
设备位置您的位置信息（仅在您授权后获取），用于显示当地天气以辅助养护建议

1.3 第三方 API 调用

当您使用植物识别功能时：

  • 植物照片会被发送到 百度AI开放平台 或 Plant.id 的服务器进行识别
  • 这些第三方服务可能有自己的隐私政策，请参阅： 百度AI开放平台、 Plant.id

2. 信息的存储与安全

2.1 本地存储

您的所有数据默认存储在设备本地：

存储位置用途
Core Data（本地数据库）植物信息、养护记录、设置偏好
Documents/PlantImages/植物照片的永久存储
Caches/PlantImages/缩略图和快速访问缓存（iOS 可自动清理）
UserDefaults用户偏好设置和本地用户标识
系统日历养护提醒日历事件（需您授权）

2.2 数据安全

  • 所有数据存储在您的设备上
  • 我们不会收集或上传您的个人数据到我们的服务器
  • 应用不包含第三方分析 SDK

3. 系统权限说明

本应用在功能需要时会请求以下权限：

权限用途是否可选
相机拍摄植物照片用于识别可选
相册选择已有照片可选
日历创建养护提醒事件可选
位置获取当地天气可选
通知发送浇水/养护提醒可选

您可以随时在 iOS 设置 > 隐私 中管理这些权限。

4. 数据保留与删除

  • 所有数据存储在设备本地，您可以通过删除应用来删除所有本地数据
  • 日历事件和本地通知需分别在系统日历和设置中手动删除
  • 我们无法访问或恢复您设备上的任何数据

5. 儿童隐私

本应用不面向 13 岁以下儿童，我们不会故意收集儿童的个人信息。

6. 隐私政策的变更

我们可能会不时更新本隐私政策。重大变更时，我们会在应用内通知您。
