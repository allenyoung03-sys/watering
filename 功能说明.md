# 植觉日记 — 智能植物养护助手

> 一款专为室内植物爱好者设计的移动植物护理伴侣。

---

## 核心功能

### 1. AI 植物识别
- 拍照识别：通过相机拍摄植物照片，使用百度 API 或 Plant.id API 自动识别植物种类
- 手动搜索：输入植物名称搜索匹配结果
- 显示植物名称、学名、置信度分数
- AI 生成养护建议（浇水/施肥/修剪/清洁频率）

### 2. 植物管理
- 添加植物：通过 AI 识别或手动添加
- 编辑植物信息：名称、学名、护理说明、房间分配
- 植物详情页：展示照片、养护状态、护理记录
- 删除植物：自动清理关联的日历事件和提醒
- 植物卡片列表：按房间筛选，今日需养护的植物置顶

### 3. 智能提醒
- 四种养护操作独立设置间隔：
  - 浇水（默认每 7 天）
  - 施肥（默认每 30 天）
  - 修剪（默认每 90 天）
  - 除虫（默认每 14 天）
- 每天指定时间发送系统通知
- 应用图标角标显示当日待养护数量
- 通知快捷操作：标记已浇水 / 推迟

### 4. 日历同步
- 自动将养护计划同步到系统日历
- 创建专属"植觉日记"日历
- 每个养护操作为 15 分钟日程事件
- 删除植物时自动清理日历事件

### 5. 养护追踪
- 一键标记养护操作完成（浇水/施肥/修剪/除虫）
- 自动更新下次养护日期
- 记录可添加文字备注和多张照片
- 查看植物完整养护历史

### 6. 时光墙
- 按日期分组的时间线视图
- 支持按时间范围、植物、养护类型、房间筛选
- 下拉刷新 + 滚动分页加载
- 添加观察记录（可关联植物、添加备注和照片）
- 支持删除养护记录

### 7. 房间管理
- 创建/重命名/删除自定义房间
- 默认房间列表（客厅、阳台、卧室等）
- 按房间筛选植物列表
- 仅空房间可删除

### 8. 用户与个性化
- 用户头像和昵称（可在设置中编辑）
- 默认提醒时间设置
- 5 页新手引导流程（功能介绍 → 权限请求 → 个性化设置）
- 通知权限状态检测
